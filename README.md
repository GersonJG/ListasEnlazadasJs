## Diferencias encontradas en Java a JS
La logica que sigue el programa respecto al funcionamiento es practicamente el mismo, solo cambiando la sintaxis y la forma en la que se llaman los diferentes metodos. En especial se diferentecia la declaracion de las variables y la forma de acceder a los datos de los nodos, pues en Java se deben declarar los constructores con los Setters and Getters para poder realizar modificaciones y claramente los metodos que permiten manipular los nodos. Mientras que por su parte en Js no se necesitan setters and getters y se puede obtener el valor directamente con los punteros, solo indicando que se modificara esa unico aspecto del la lista o nodo que se trabaja, por lo que se utiliza mucho la funcion de *"this."* para indicar el argumento a utilizarse y no modificar la estructura original de los listas que obtiene cada metodo.
